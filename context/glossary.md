# Glossary

- **Goal**: high-level objective containing multiple tasks.
- **Task**: executable unit of work linked to a goal.
- **Plan Mode**: discovery/spec generation mode.
- **Execute Mode**: task/goal execution mode.
- **Sub-chat**: chat thread within a parent chat, with mode/cli/session state.
- **Worktree**: git-isolated execution path for parallel agent work.
- **Checkpoint (Entire)**: commit/session context evidence for a run.
